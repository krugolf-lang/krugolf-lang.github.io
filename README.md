# ครูกอล์ฟ (Krugolf) — เว็บไซต์บทเรียนออนไลน์

เว็บไซต์ต้นแบบ (prototype) สอนเขียนโปรแกรม & ทักษะดิจิทัลสำหรับเด็กไทย
สร้างด้วย React 18 + Babel (in-browser) ไม่ต้องติดตั้งอะไร แค่เปิดไฟล์ก็ใช้งานได้

## วิธีเปิดใช้งาน
เปิด `index.html` ผ่านเว็บเซิร์ฟเวอร์ (ไม่ใช่ดับเบิลคลิกตรง ๆ เพราะมีการโหลดไฟล์ .jsx)
ตัวอย่างรันเซิร์ฟเวอร์ง่าย ๆ:

```bash
# Python
python3 -m http.server 8000
# หรือ Node
npx serve .
```
แล้วเปิด http://localhost:8000

> หมายเหตุ: เวอร์ชันไฟล์เดียว (standalone) เปิดด้วยดับเบิลคลิกได้เลย แต่ชุดไฟล์แยกนี้ต้องรันผ่านเซิร์ฟเวอร์

## โครงสร้างไฟล์
| ไฟล์ | หน้าที่ |
|------|---------|
| `index.html` | ไฟล์หลัก โหลด React, Babel, ฟอนต์ และสคริปต์ทั้งหมด |
| `styles.css` | Design tokens (สี/ฟอนต์/เงา/ระยะ) + สไตล์ส่วนกลาง + responsive |
| `ui.jsx` | ไอคอน, โลโก้, primitives (Badge/Stat/Progress/Motif) และ **ข้อมูลคอร์ส (COURSES)** |
| `landing.jsx` | หน้าแรก (Hero, ขั้นตอนการเรียน, คอร์สแนะนำ, CTA) |
| `catalog.jsx` | CourseCard, หน้ารวมคอร์ส (ค้นหา/กรอง), หน้ารายละเอียดคอร์ส |
| `lesson.jsx` | หน้าเรียน/เครื่องเล่นวิดีโอ + playlist + progress |
| `app.jsx` | Nav, Footer, ระบบ router (state) และ mount แอป |

## การแก้ไขที่พบบ่อย
- **เพิ่ม/แก้คอร์ส:** แก้ array `COURSES` ใน `ui.jsx`
- **เปลี่ยนสีธีม:** แก้ตัวแปร CSS ใน `:root` ที่ `styles.css` (เช่น `--coral`, `--bg`)
- **เปลี่ยนฟอนต์:** แก้ `@import` และตัวแปร `--sans/--loop/--mono` ใน `styles.css`

## หมายเหตุสำหรับโปรดักชัน
ตอนนี้ใช้ Babel แปลง JSX ในเบราว์เซอร์ (เหมาะกับ prototype)
หากนำไปใช้จริง ควร precompile ด้วย Vite/Next.js เพื่อประสิทธิภาพ
